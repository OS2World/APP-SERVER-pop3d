This POP3 daemon is derived from Linux sources
for pop3d 1.006c. The sources are changed significantly,
so the number of this version is 1.010.

System requirements:
OS/2 Warp 3.0 or higher
TCP/IP v.4.0 or 4.1.
EMX 0.9d runtime.
